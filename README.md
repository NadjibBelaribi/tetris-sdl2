# Tetris NH

Master Project 
Advanced programming with C++ and SDL2

# Build It ? 

- chmod +x run.sh
- ./run.sh 
  
# Use it ?

After lanching it : 
- Press "S" for solo mode
- Press "M" for duo mode
- Game is ON
- Use the directions keys to manipulate the tetrominos
- When a game is done , Press "Y" to replay , else Press "N"
